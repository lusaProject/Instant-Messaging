即时通讯 (Instant Messaging, IM) 简介
即时通讯（IM）是一种基于 互联网 的通信服务，它允许用户在计算机、智能手机或其他设备上，以 近乎实时 的速度进行双向、互动式的交流。IM 最核心的特征是其“即时性”和“互动性”，它极大地模拟了人们面对面交流的效率和体验。

核心功能与特点
文本聊天: 最基本和最常用的功能，用户可以发送和接收文字信息。

多媒体传输: 支持发送图片、音频、视频、文档和位置信息等多种类型的文件。

语音和视频通话: 允许用户进行实时的“点对点”或“多方”语音和视频交流。

群组聊天: 用户可以创建群组，方便多人同时进行讨论和协作。

状态显示: 通常会显示用户的在线状态（如“在线”、“忙碌”、“隐身”等），帮助判断是否适合联系。
